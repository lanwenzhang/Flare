# Hybrid Renderer

## Features
- GPU-driven rendering
  - Frustum culling
  - Occlusion culling
  - Indirect draw
- Deferred shading
- Indirect lighting
  - Dynamic diffuse global illumination
  - Screen space reflection
- Shadow
  - Ray-traced shadow
- Post processing
  - Temporal anti-aliasing
  - ACES filmic tone mapping
- UI control
