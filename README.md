# Hybrid Renderer

## Features
- GPU-driven rendering
  - Frustum culling
  - Occlusion culling
  - Indirect draw
- Deferred shading
- Physically based rendering
  - Lambert diffuse BRDF
  - Microfacet Cook-Torrance BRDF
- Indirect lighting
  - Dynamic diffuse global illumination
  - Screen space reflection
- Shadow
  - Ray-traced shadow
- Post processing
  - Temporal anti-aliasing
  - SVGF denoising
  - ACES filmic tone mapping
- Imgui
  - Interactive control
  - GPU profiler
